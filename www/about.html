About Webserv

What is Webserv?

Webserv is a HTTP/1.1 server implementation written in C++98. It's designed to handle multiple simultaneous connections efficiently using epoll-based I/O multiplexing.

Key Features

  • ✅ HTTP/1.1 Protocol Support
  • ✅ GET, POST, DELETE Methods
  • ✅ Static File Serving
  • ✅ CGI Script Execution
  • ✅ File Upload (multipart/form-data)
  • ✅ Virtual Host Support
  • ✅ Custom Error Pages
  • ✅ Directory Listing
  • ✅ Non-blocking I/O

Technical Details

Language: C++98
I/O Model: epoll (Linux)
Protocol: HTTP/1.1
Max Connections: 1024

Architecture

The server uses a modular design with separate handlers for:

  • RequestHandler: Routes requests to appropriate handlers
  • FileHandler: Serves static files
  • UploadHandler: Processes file uploads
  • CgiHandler: Executes CGI scripts
  • ErrorHandler: Generates error pages
